# Changelog — Meridian Commerce Platform v4.2.0

## Release Overview
On 14 October 2024, we deployed v4.2.0 to production across all regions. This release focuses on checkout resilience, merchant analytics, and backend scalability ahead of the holiday season. Engineering lead Sarah Chen oversaw the rollout, which completed with zero downtime across our North American and EU clusters.

## Performance & Infrastructure
We rebuilt the product catalog indexer in Rust, reducing average query latency from 340 ms to 85 ms. The new pipeline handles 12,000 requests per second at peak load without degradation. We also added redundant failover nodes in Frankfurt and Singapore.

- Improved CDN cache hit rates by 28% using edge-computed image variants
- Reduced database connection pooling overhead by 40%
- Migrated legacy cart services to Kubernetes 1.29
- Added automated rollback triggers if error rates exceed 0.5%

## Checkout & Payments
The payment gateway now supports 3D Secure 2.3 for Visa and Mastercard transactions. We integrated Stripe Link and Apple Pay Express, cutting average checkout time to 42 seconds. Refund processing was streamlined: merchants can now issue partial refunds up to $2,500 directly from the dashboard without contacting support.

## Merchant Tools
Dashboard analytics received a major upgrade. Merchants can export reports in CSV, JSON, or PDF formats dated back to 1 January 2023. The inventory forecaster uses machine learning to predict stock-outs 14 days in advance. We also introduced role-based access control with five default permission tiers.

## Bug Fixes & Stability
We resolved 87 reported issues, including a race condition in multi-currency pricing and an intermittent 502 error on the mobile web cart page. Tax calculation for Canadian provinces was corrected for Quebec and British Columbia. All fixes were verified through automated regression testing before merge.
