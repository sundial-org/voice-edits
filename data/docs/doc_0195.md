# Technical Design Document: Echoes of Aethelgard

## Project Overview
Echoes of Aethelgard is a 2D narrative-action platformer developed by Mosslight Studios. The game targets a Q3 2025 release window on Steam and Nintendo Switch. This document outlines core systems, art pipeline, and production milestones for the six-person team led by creative director Lena Voss.

## Core Gameplay Systems
The player controls a relic hunter navigating decaying sky-archipelagos. Movement emphasizes momentum-based gliding and wall-bouncing rather than traditional double-jumps. Combat uses a single modular weapon, the Aether-Resonator, which can be retuned at shrines.

Key mechanics include:
- Resonance charging: hold for 1.8 seconds to emit a shockwave that breaks brittle platforms
- Gliding decay: lift reduces by 12% per second to prevent infinite flight
- Shrine checkpoints: only three per level, forcing risk-reward routing
- Echo fragments: 42 collectibles that unlock alternate dialogue trees

## Art and Audio Pipeline
Visuals combine hand-painted backgrounds with vector-animated characters at 60 fps. Background layers parallax at 0.4x, 0.7x, and 1.0x speeds relative to the camera. Sound design by composer Jiro Tanaka uses a hybrid orchestral-electronic palette with dynamic stems that crossfade based on player altitude.

Asset budgets are tight: 14 unique enemy types, 6 biomes, and 3 boss encounters. All sprites are authored in Aseprite and exported through a custom Python pipeline that validates palette constraints automatically.

## Technical Architecture
The game runs on a custom C++ engine with Lua scripting for level logic. Target frame time is 16.67 ms on Switch handheld mode. We use object pooling for projectiles and a deterministic replay system for speedrunning validation.

Networking is absent; saves are stored locally using JSON with AES-128 encryption. The build pipeline runs on GitHub Actions with automated playtesting reports generated every night at 02:00 UTC.

## Milestones and Deliverables
Alpha build with three completed biomes is due March 14, 2025. Beta feature-complete lock is scheduled for June 2, 2025, followed by four weeks of polish and localization into Japanese and Portuguese.

Risk items include Switch performance in the Crystal Spire biome and animation sync for the final boss. Weekly playtests with our external group of 18 testers begin February 3, 2025.
