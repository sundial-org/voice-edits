# MerchantOS Changelog — Q1 2024 Release

## Overview
On March 14, 2024, we deployed MerchantOS v3.8 across our global clusters, marking the largest platform update since the November 2023 infrastructure migration. This release targets checkout reliability, catalog scalability, and merchant-facing analytics. Over 14,000 active storefronts will receive the update automatically with zero downtime. Engineering lead Sarah Chen and the Commerce Reliability team coordinated the rollout across three regions: us-east-1, eu-west-2, and ap-southeast-1.

## Checkout & Payments
We rebuilt the payment orchestration layer to reduce gateway timeouts by 62%. Merchants using Stripe, Adyen, or Braintree will see faster confirmation screens and fewer abandoned carts.

Key improvements include:
- Real-time tax calculation via the new TaxEngine v2 module
- Support for 3D Secure 2.2 authentication on Visa and Mastercard transactions
- One-click refund processing for orders placed within the last 90 days
- Improved error messages when a customer’s bank declines a charge
- Dynamic currency conversion for 28 supported fiat currencies

The average checkout completion time dropped from 4.2 seconds to 1.8 seconds in our load tests.

## Inventory & Catalog Management
Bulk editing is now available for catalogs exceeding 50,000 SKUs. Warehouse managers can update pricing, tags, and stock counts across multiple warehouses without triggering legacy sync conflicts. We also introduced predictive restocking alerts based on 30-day sales velocity rather than static thresholds.

Name changes: the "Warehouses" tab has been renamed to "Fulfillment Centers" to align with multi-node logistics partners like ShipBob and Deliverr.

## Security & Performance
We rotated all API keys for the merchant dashboard and enforced TLS 1.3 across every endpoint. The new Web Application Firewall rules blocked 1.2 million malicious requests during the March 12 canary deployment. Database query latency for order lookups improved by 0.4 seconds on average, and CDN caching now covers 98% of product image requests.

## Bug Fixes
- Fixed an issue where discount codes expired 24 hours earlier than configured for UTC-5 merchants
- Resolved PDF invoice corruption on mobile Safari browsers running iOS 17.3
- Corrected shipping rate miscalculations for packages over 25 kg in the Nordic region
- Restored webhook delivery for inventory low-stock events that failed silently since February

All fixes are live in production. Merchants should clear their browser cache if the dashboard layout appears misaligned. For support, contact the platform team with release ticket OS-3884.
