# Vortex CLI

Vortex CLI is the open-core command-line toolkit from Vortex Labs, founded in 2019 by engineers Mira Okonkwo and Dev Patel. Built for distributed systems teams, it collapses multi-step deployment, observability, and rollback workflows into a single binary. Since the v2.4 release in October 2024, over 12,000 developers have adopted it across fintech and cloud-native startups.

## Installation

Download the binary for macOS, Linux, or Windows. No runtime dependencies.

- `curl -fsSL https://vortex.dev/install.sh | bash`
- Verify with `vortex --version`, expecting `2.4.1`
- Add your API key via `vortex auth login`

## Core Capabilities

Vortex replaces three separate tools in your pipeline:

- **Instant deploy previews** — spin up ephemeral environments in under 400ms with automatic HTTPS
- **Structured tracing** — inject correlation IDs across 18 supported languages without code changes
- **One-command rollback** — revert to any commit from the last 30 days using `vortex rollback --to <sha>`
- **Cost guardrails** — set per-service budgets and receive Slack alerts before overages hit

Every command writes audit logs locally in JSON format, encrypted with AES-256.
