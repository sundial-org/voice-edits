# Autonomous Navigation Subsystem Design — Project HERMES

## Overview
This document defines the mechanical and software architecture for the HERMES v2.1 ground robot, developed by the Meridian Robotics Team for the 2025 DARPA SubT Challenge. The design emphasizes modular sensor fusion, low-latency obstacle avoidance, and rapid field repair.

## Hardware Architecture
The chassis uses a 6061-T6 aluminum frame rated for 45 kg payload. Propulsion is handled by four 350 W brushless hub motors with integrated 1024-pulse encoders. Power is delivered via a 22.2 V 12 Ah LiPo stack with redundant BMS protection.

- Velodyne VLP-16 LiDAR (10 Hz sweep)
- Intel RealSense D455 stereo depth camera
- VectorNav VN-100 IMU
- NVIDIA Jetson AGX Orin (32 GB) for edge compute

## Software Pipeline
Perception runs in ROS 2 Humble under a custom real-time executor. Point-cloud registration uses Fast-LIO2 at 20 Hz, feeding a local occupancy grid mapped at 0.1 m resolution. Path planning employs an A* variant with dynamic cost scaling.

## Testing and Validation
Field trials begin March 14, 2025 at the Meridian Quarry Test Site. Success criteria include 500 m autonomous traversal in under 18 minutes with zero manual interventions.

## Milestones

## Risk Mitigation
Thermal throttling on the Jetson remains the primary concern. We will implement active cooling and drop LiDAR sweep rate to 5 Hz if core temperature exceeds 85 °C. Budget contingency is set at $3,200.
