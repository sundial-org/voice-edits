# Meridian Health IT Infrastructure Readme

This repository documents the core network, endpoint, and clinical system configurations managed by the Meridian Health IT Operations group. All changes require approval through the Change Advisory Board and must align with HIPAA security controls.

## Scope and Governance

The documentation covers three campuses: Riverside General, North Hills Clinic, and the downtown Data Center. Dr. Elena Vasquez serves as Chief Information Officer, with primary system ownership assigned to the Network Engineering team led by Marcus Chen. Any modifications to VLANs, firewall rules, or EHR integrations must be documented within 24 hours of deployment.

## Core Systems Inventory

Our production environment supports 1,247 clinical workstations, 842 IoT medical devices, and 34 virtual servers hosting the Epic EHR instance version 2024.08. Key platforms include:

- Riverside PACS imaging cluster (storage: 480 TB)
- North Hills telemedicine gateway (uptime target: 99.99%)
- Central identity provider running Active Directory 2019
- Backup vault with 90-day retention and quarterly restoration tests

## Change Windows and Maintenance

Standard maintenance occurs every Saturday from 02:00 to 06:00 EST. Emergency patches require on-call approval from the senior engineer roster. The 2025 Q1 upgrade cycle will replace core switches at Riverside General and migrate the pharmacy database to the new cluster by March 15, 2025. All maintenance logs are stored in this repository under /logs/.

## Security and Compliance

Access to this repository is restricted to staff with valid Meridian Health credentials and multifactor authentication enabled. Annual audits are conducted in partnership with CyberTrust Consulting, with the next review scheduled for November 12, 2025. Incident reports must reference ticket numbers in the ServiceNow portal.

## Contact and Escalation

For urgent outages affecting patient care systems, page the on-call engineer via the Opsgenie rotation. Non-urgent requests should be submitted through the internal portal or emailed to it-ops@meridianhealth.org. Response time targets are 15 minutes for Severity 1 events and four hours for standard requests. Thank you for maintaining the integrity and availability of our healthcare technology environment.
