# Collective Notes: Signal Flow & Session Discipline

## Room Calibration
We met at the warehouse on March 14 to recalibrate Studio B. Marcus Chen brought the measurement mic and ran a sweep at 85 dB SPL. The null at 128 Hz was worse than last quarter, so we moved the rear bass trap 18 inches left and added a second cloud above the mix position. Before touching any fader, confirm the monitoring chain is flat. If the room lies, the mix lies.

## The Chain
Our agreed input path remains simple but strict. Mic pre into the 1073 clone, then a single stage of 1176-style compression with no more than 4 dB of gain reduction. From there, hit the converters at -18 dBFS peak to leave headroom for the bus. Do not track hot just because the meter glows red. The collective standard is 48 kHz / 24-bit unless a client specifically requests archival at 96 kHz.

## Session Protocol
Discipline saves money. Label every track with the performer name, instrument, and date recorded. Bounce stems with 2-second handles and include a reference mix at -14 LUFS integrated. Before anyone hits export, run the checklist:

- All mute automation disabled?
- Crossfades longer than 6 ms?
- Print track named with version number?
- Reference mix printed to same folder?
- Backup copied to the NAS before leaving?

If any box is unchecked, the session is not done.

## Reference Mixes
We compared three releases this month: Lena Voss’s *Midnight Static*, the collective’s 2023 compilation *Concrete Gardens*, and the demo Marcus tracked on February 2. The Voss record sits wider but drier; ours had too much 400 Hz mud in the guitar bus. Use references not to copy, but to check translation. Listen on headphones, in the car, and on phone speakers. If the vocal disappears anywhere, fix it in the source, not with a plugin.

## Takeaway Checklist
Keep notes in the shared log after every session. Note what worked, what failed, and any gear that needs service. The next engineer should not have to guess. Our reputation is built on consistency, not luck. See you Thursday.
